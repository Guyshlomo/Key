# Key Community - React Native App

אפליקציית React Native עם Expo Go

## התקנה

1. התקן את התלויות:
```bash
npm install
```

או

```bash
yarn install
```

## הרצה

### עם Expo Go

1. התקן את אפליקציית Expo Go על הטלפון שלך:
   - [iOS App Store](https://apps.apple.com/app/expo-go/id982107779)
   - [Google Play Store](https://play.google.com/store/apps/details?id=host.exp.exponent)

2. הפעל את השרת:
```bash
npm start
```

3. סרוק את ה-QR code עם:
   - **iOS**: מצלמה של iPhone
   - **Android**: אפליקציית Expo Go

### עם סימולטור

**iOS Simulator (רק macOS):**
```bash
npm run ios
```

**Android Emulator:**
```bash
npm run android
```

## טכנולוגיות

- React Native
- Expo SDK ~51.0.0
- React 18.2.0
